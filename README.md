**F1-Race Winner Prediction System**
An End-to-End Machine Learning pipeline for predicting Formula-1 race outcomes using historical data from multiple sources.

## Features

- Data Collections from multiple F1 data sources (Ergast API, Official F1 Site, Weather Data)
- Comprehensive data processing and feature engineering pipeline
- Multiple prediction models:
    - Race winner prediction
    - Podium finish prediction
    - Driver performance relative to teammates
    - Qualifying position prediction

- Model evaluation and performance tracking
- (Future) Interactive dashboard for visualizing predictions
